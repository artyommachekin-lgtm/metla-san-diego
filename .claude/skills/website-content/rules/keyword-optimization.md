# Keyword Optimization Workflow

## 1. Primary Keyword Strategy
- **Primary keyword**: "house cleaning san diego" — target in EVERY piece of content
- **Placement rule**: Primary keyword MUST appear in title tag, H1, first 100 words, and meta description
- **Density cap**: Max 3 uses of exact-match primary keyword per 1500 words. Use variations for the rest.

## 2. Keyword Variations (use these to avoid stuffing)
- "house cleaning services in san diego"
- "san diego house cleaning"
- "professional house cleaners san diego"
- "residential cleaning san diego"
- "maid service san diego"
- "home cleaning san diego"
- "cleaning services san diego"

## 3. Service-Specific Keywords (use on matching service/blog pages)
- Standard: "regular house cleaning san diego", "biweekly maid service san diego", "weekly cleaning service"
- Deep: "deep cleaning san diego", "whole house deep clean san diego", "first time deep cleaning san diego"
- Move: "move in cleaning san diego", "move out cleaning san diego", "end of lease cleaning san diego"
- Post-construction: "post construction cleaning san diego", "post renovation cleaning san diego"
- Vacation rental: "airbnb cleaning san diego", "VRBO cleaning san diego", "vacation rental turnover cleaning"

## 4. Location Keywords (use on matching location/blog pages)
Format: "[service] [neighborhood]" — e.g., "house cleaning la jolla", "maid service north park"
Always include the ZIP code naturally in body text.

## 5. Anti-Stuffing Rules
- NEVER repeat the exact same keyword phrase in consecutive sentences
- NEVER use more than 1 keyword per sentence
- Every keyword placement must read naturally — if it sounds forced, rewrite the sentence
- Use pronouns ("we", "our team") and natural references ("the cleaning", "this service") between keyword mentions
- Headers: Only 50% of H2s should contain the primary keyword; the rest use variations or topic-specific terms

## 6. Cannibalization Prevention
Before writing any new blog post:
1. Check `data/blog-posts.ts` for existing posts with overlapping topics
2. If overlap exists, differentiate by: location specificity, angle (seasonal vs. general), or service type
3. The NEW post title must include the neighborhood name; the EXISTING generic post keeps its broader title
4. Meta descriptions must use completely different phrasing — never share more than 3 consecutive words

## 7. Human Readability Check
After writing, verify:
- [ ] Would a real San Diego homeowner share this on Facebook? If not, rewrite.
- [ ] Does every paragraph answer a question or provide actionable advice?
- [ ] Are there any sections that feel like "SEO filler"? Remove them.
- [ ] Read the first sentence of each section aloud — does it sound natural?
